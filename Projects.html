Caesar Ciphers - Python & Matlab
  • GitHub

I created two simple ciphers both in Matlab, and Python. A Caesar Cipher simply encrypts a message by switching a letter with another letter a set length down the alphabet, and doing this for each letter in a message. The Matlab script only encrypts the message at an inputted length, and the Python script encrypts and decrypts a message at 13 letters down the alphabet.

Collatz Sequence Generator - Python
  • GitHub

The Collatz Sequence is essentially a math sequence that always result in 1. From this sequence, a tag system was created. In which any amount of A's are inputted, and it always results in 1 A. The program prints out the entire history of the sequence.

Verify Magic Array - Matlab
  • GitHub

A Magic Array is simply an array in which every row and column adds to the same value. I created a small function to ensure if an Array is a magic array in Matlab, so that it doesnt need to be checked manually.
